# Personal Website

Static HTML/CSS/JS personal site — portfolio, work experience, projects, and contact info.

Deployed via GitHub Pages at a custom domain (see `CNAME`).

## Structure

- `index.html` — about/bio, work experience, featured-projects teaser, contact links
- `projects.html` — full project list
- `assets/css/styles.css` — design tokens + base + component styles
- `assets/js/main.js` — nav toggle, smooth scroll, scroll-reveal
- `404.html` — custom not-found page

## Local preview

Serve with any static file server, e.g.:

```
python -m http.server 8000
```

Then open http://localhost:8000
